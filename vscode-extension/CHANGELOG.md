# Changelog

## 1.0.1

- **Comment Preservation**: The formatter now correctly preserves and aligns comments within factory constructors.
- **Improved Path Resolution**: The extension now prefers `dart run` and handles project roots more intelligently.
- **Auto-Installation**: Added a prompt to automatically add `freezed_go_style` to `dev_dependencies` if missing.
- **Logging**: Added an Output Channel for better debugging.
- **Bug Fixes**: Resolved issues with Dart SDK path resolution when running from inside `bin/`.

## 1.0.0

- Initial release of the VS Code extension.
